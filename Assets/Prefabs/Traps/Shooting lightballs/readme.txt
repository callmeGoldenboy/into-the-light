Description: 
A point that shoots out light balls with the help of a script called
"shootLightBalls"

Public variables:
(Light ball ,Already set in the prefab, this is what it shoots out)
NumberOfBalls -> how many balls to shoot out
Strength -> ball speed strength
IntervalSeconds -> How many seconds for next shooting phase


Dependencies: 
No dependencies you have to think of, but in lightball there is 
a need to make the ground blocks with tag "ground" in order for
the lightball to destruct on collison.

How to use:
Drag and drop. There is already default values for the variables.
