Description: an object used to handle audio in the game

Public variables: 
size = number of audio objects you want to play (size of the array containing this elements)
name = and id used to identify which audio to play out of all in the array
clip = refernce to the sound that is going to play
rest of the variables are used to adjust setting to fit your desires


Dependencies: audiomanager script and soundhandler script


How to use: drag and drop the audiomanager prefab, adjust the setting as desired. If it is for the castle
            theme the variable castle music must be checked, for the forest theme it must be unchecked.
